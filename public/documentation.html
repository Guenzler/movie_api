About this Movie API

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL and query parameters HTTP method Request body data format Response body data format
Get a list of all movies /movies GET None An array of JSON objects holding data about all movies in the database
Get data about a single movie by title /movies/[title] GET None A JSON object holding data about a single movie, containing title, description, genre, director and image URL
Example: { "Genre": { "Name": "Drama", "Description": "sample description" }, "Director": { "Name": "Michael Curtiz", "Bio": "Example text", "Birthdate": "1886-12-24T00:00:00.000Z", "Deathdate": "1962-04-10T00:00:00.000Z" }, "_id": "0908098988098089", "Title": "Casablanca", "Description": "Sample description", "ImagePath": "sampleurl.jpg", "Featured": false }
Get description of genre by name/title of genre /movies/genre/[genreName] GET None A JSON object holding data about a single genre, containing genre name, description. Example: { "Name": "Drama", "Description": "Example description" }
Get data about a director by name /movies/directors/[directorName] GET None A JSON object holding data about a single director, containing director name, bio, birth year and death year. Example: { "Name": "Charles Chaplin", "Bio": "Sample text", "Birthdate": "1889-04-16T00:00:00.000Z", "Deathdate": "1977-12-25T00:00:00.000Z" }
Allow new users to register /users POST A JSON object holding data about the user to add, structured like: { "username": "Patricia", "password": "somePassword", "email": "someEmail", "birthdate": "2001-08-01" }
required fields: username, password and email 		A JSON object holding data about the user that was added, including ID. Example: { "username": "Patricia", "password": "somePassword", "email": "someEmail", "birthdate": "2001-08-01T00:00:00.000Z", "favoriteMovies": [], "_id": "ce3d7bcb-952b-498d-" }
